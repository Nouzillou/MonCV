Mes différents projets :

Cette page présente plusieurs projets que j’ai réalisés au cours de ma formation. Ils m’ont permis de développer mes compétences en programmation, en développement web et en ingénierie.

Répertoire Téléphonique (NSI-1ère)

Dans le cadre de la spécialité Numérique et Sciences Informatiques (NSI), j’ai développé un programme permettant de gérer un répertoire téléphonique. Le projet consistait à créer une application capable d’ajouter, modifier, rechercher et supprimer des contacts dans une base de données simple (modélisée par un fichier texte). Ce projet m’a permis de :

  • apprendre à structurer un programme
  • manipuler des données
  • travailler la logique algorithmique
  • utiliser le langage Python et la bibliothèque Flask
Ce travail m’a aidé à mieux comprendre les bases de la programmation et la gestion de données dans une application.


Site web sur Tim Berners-Lee (Projet NSI – Première)

Lors de ma spécialité NSI, j’ai également réalisé un site web présentant Tim Berners-Lee, l’inventeur du World Wide Web. L’objectif était de créer un site informatif expliquant :

  • sa biographie
  • la création du Web
  • son impact sur Internet aujourd’hui
Pour ce projet, j’ai utilisé :
  • HTML
  • CSS
Ce travail m’a permis de découvrir les bases du développement web et la structure d’un site internet.

Projet robot (Sciences de l’ingénieur – ENSIBS)

Durant ma première année de PEI à l’ENSIBS, j’ai participé à un projet de conception et de programmation d’un robot dans le cadre des sciences de l’ingénieur. Ce projet consistait à :

  • comprendre le fonctionnement d’un système robotisé
  • concevoir une solution technique
  • programmer le robot pour réaliser certaines actions avec des capteurs
Ce travail m’a permis de développer :
  • mes compétences en travail d’équipe
  • ma compréhension des systèmes techniques
  • mes bases en programmation et en ingénierie

Création de mon site CV (Projet actuel)

Dans le cadre de ma formation, j’ai réalisé ce site web afin de présenter mon parcours, mes compétences et mes projets. L’objectif est de créer un CV numérique interactif permettant de valoriser mon profil et de présenter mon orientation professionnelle. Pour ce projet, j’utilise :

  • des principes de design web et d’organisation de contenu
Ce site me permet également de continuer à développer mes compétences en développement web.
